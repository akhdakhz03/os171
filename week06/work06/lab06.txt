1. a. what are PID and PPID ?
-PID 		: ketika process berjalan, maka ia akan menghasilkan nomor unik yang bernama Process ID (PID)
-PPID		: PID dari process yang berjalan pada parent.

b. what do the getpid(), getppid(), sleep, and wait functions do?
-getpid()	: mengembalikan ID proses dari proses pemanggilan
-getppid() 	: mengembalikan ID proses parent dari proses pemanggilan
-sleep	  	: menunda eksekusi program untuk sementara waktu
-wait	  	: menunda eksekusi dari proses yang sedang berlangsung  sampai salah satu proses anak selesai(terminates)

c. what are the return values of fork() ?
- Output dari fork() method akan menghasilkan proses baru, dimana proses baru tersebut disebut sebagai child process sementara proses pemanggilannya disebut sebagai parent process. Serta mengembalikan ID proses dari proses anak kepada proses induk. Value merupakan ID proses yang dikembalikan oleh child proses, jika value > 0 tandanya ada proses baru yang diciptakan, jika value < 0 berarti tidak ada proses baru yang diciptakan.

2.  procStatus 		: mengambil nilai level, PID, dan PPID dari task yang sedang diproses dan selanjutnya menampilkannya ke layar
   - levelFork 		: melakukan proses fork()
   - wait(NULL) 	: agar proses parent tidak mati karena menunggu proses child selesai dieksekusi(terminates)
   - fflush(NULL)	: fungsi ini tidak memberikan pengaruh pada program.
