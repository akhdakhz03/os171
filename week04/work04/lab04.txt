1. (a) Perbedaan antara kode program pada global-char dan local-char adalah 
kode program pada global-char deklarasi variable berada di luar fungsi
sementara kode program pada local-char variable berada di dalam fungsi.
perbedaan output dari global-char dan local-char ada pada pengalokasian 
alamat variabel pada memori.

(b) global variable tidak membutuhkan alamat memori main. 
local variable membutuhkan alamat memori dari fungsi dimana dia berada. 

(c) variabel global adalah variabel yang di deklarasikan di luar fungsi, 
dan dapat diakses oleh semua fungsi di dalam program tersebut.
variabel lokal adalah variabel yang di deklarasikan di dalam fungsi, dan 
hanya dapat dipakai di dalam fungsi tersebut.
source : (funprogramming.org/50-What-are-global-and-local-variables.html)

O_CREATE yang berguna untuk membuat file baru jika file tidak ada atau menerima nilai dari O_RDWR yang berguna untuk membuka file sehingga dapat dibaca dan ditulis (read and write).
(b). digunakan untuk menginputkan nilai yang digunakan sebagai hak akses pada file tersebut. S_IRWXU maka user memiliki hak akses untuk (read, write dan execute), 
0711
(c).  file open and close I/O yang meliputi open, close, read, write dan append.
fungsi standar yang digunakan oleh sistem operasi untuk memberikan akses kepada
suatu file.
