- 01-client.c : Ketika program dijalankan didalam terminal, terminal akan bertindak sebagai terminal client. Berikutnya agar dapat terhubung dengan terminal server, terminal client harus menyesuaikan port dengan terminal server dan 

		menuliskan IP yang sedang digunakan. Setelah terhubung, berikutnya terminal client bisa mengirim pesan kepada terminal server.



- 02-clisvr.c : Ketika program dijalankan, selanjutnya program akan terhubung ke beberapa termnial dengan memasukan IP dari masing-masing terminal(komputer) yang lain. Setelah semua terminal terhubung, selanjutnya kompuer yang pertama 

		akan memanggil IP komputer yang terhubung terakhir. Setelah itu akan muncul pesan 'Hallo' pada jendela masing-masing terminal.
