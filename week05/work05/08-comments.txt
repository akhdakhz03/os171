5. system() merupakan library yang berfungsi untuk menerima suatu command linux untuk di eksekusi oleh command proses dan ditampilkan sebagaimana command tersebut dijalankan pada lingkungan linux.

6. baris kode yang berfungsi untuk menampilkan perintah adalah
sprintf(strSYS2, "top -b -n 1 -p%d | tail -5", myPID);
sprintf(strSYS1, "top -b -n 1 -p%d | tail -1", myPID);

7. pada top -h -n l -p%d | tail -5/-1
baris kode ini mengeksekusi command top dengan beberapa key yang berfungsi sebagai berikut :
-b    :  pada mode ini command top akan berjalan hingga iterasi berakhir.
         command ini dikombinasikan dengan command -n untuk membatasi berapa jumlah iterasi yang akan terjadi.
-n 1  :  jumlah iterasi maksimal nya adalah 1
-p%d  :  memonitor proses dengan spesifik PID yang di lewatkan melalui 
         variabel myPID.
